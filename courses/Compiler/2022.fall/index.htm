Background:

As software becomes complicated, modern software products need integrated design consideration in CPU architecture, compilation, algorithms, and user-experience to achieve competitive performance.  In such consideration, the central position is taken by compiler technology.  In fact which has also affected and inspired the development of CPU architecture, and vice versa.  To achieve the best hardware performance, we need understand what kinds of machine instructions are generated by common programming language structures.  The technology of document analysis and execution code optimization is also important in preparing and inspiring students for developing various software. 

 

Goals:

Preparing students for the following fundamentals of compilers.

l  lexical analysis

l  parsing

l  semantics analysis

l  code generation

l  code optimization

Moreover, we will have the following arrangement:

l  survey assignments so that students can access to the cutting edge research topics.

l  experiments with compilers via projects.
